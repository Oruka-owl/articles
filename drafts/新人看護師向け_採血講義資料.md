## 1. 採血の目的と種類

### 目的
採血は診断・治療の補助として行われる最も頻度の高い看護技術のひとつです。

| 目的 | 具体例 |
|------|--------|
| 診断 | 血液検査（CBC、生化学、感染症など） |
| 治療効果の判定 | 投薬後の血中濃度測定 |
| 病態の把握 | 電解質バランス、腎機能、肝機能 |
| 輸血前検査 | 血液型・交差適合試験 |

### 採血の種類

| 種類 | 主な用途 | 穿刺部位 |
|------|----------|----------|
| 静脈血採血 | 一般的な血液検査 | 肘正中皮静脈など |
| 動脈血採血 | 血液ガス分析 | 橈骨動脈、大腿動脈 |
| 毛細血管血採血 | 血糖測定（簡易） | 指先、耳垂 |

> **本講義では最も頻度の高い「静脈血採血」を中心に解説します。**

## 2. 解剖学的基礎知識

### 採血に使用する主な静脈

```
上腕内側面の静脈分布（右腕・前面）

    ─── 橈側皮静脈（Cephalic vein）
   /
  肘窩
   \
    ─── 尺側皮静脈（Basilic vein）
        │
        肘正中皮静脈（Median cubital vein）← 最もよく使用
```

#### 各静脈の特徴

| 静脈名 | 特徴 | 注意点 |
|--------|------|--------|
| **肘正中皮静脈** | 太く走行が安定。第一選択 | 個人差あり |
| **橈側皮静脈** | 橈骨側を走行。比較的見えやすい | 細い場合がある |
| **尺側皮静脈** | 尺骨側を走行 | 尺側に正中神経・上腕動脈が近接。損傷リスク注意 |
| **前腕静脈** | 肘が使えない場合の代替 | 細く固定しにくい |
| **手背静脈** | 高齢者などで使用 | 痛みが強い、固定しにくい |

### 避けるべき部位
- 乳房切除後の患側上肢（リンパ浮腫リスク）
- シャント造設側の上肢（透析患者）
- 点滴・輸液中の同側上肢
- 皮膚病変・感染・血腫のある部位
- 麻痺側（疼痛を訴えられない）

## 3. 必要物品の準備

### 基本セット

```
[ 採血トレイ ]
 ├── 採血管（目的に応じた種類・本数）
 ├── 採血針（または真空採血ホルダー付き針）
 ├── 駆血帯（ターニケット）
 ├── アルコール綿（消毒綿）
 ├── 滅菌ガーゼまたは綿球
 ├── 絆創膏
 ├── 手袋（清潔）
 ├── 針捨て容器（耐貫通性廃棄ボックス）
 └── ラベル（患者氏名・ID・採血管の種類が確認済みのもの）
```

### 採血管の色と用途

| キャップの色 | 抗凝固剤・添加物 | 主な検査 |
|-------------|----------------|----------|
| 赤（なし） | なし | 生化学、免疫血清 |
| 紫（EDTA） | EDTA-2K/2Na | 血算（CBC）、血液型 |
| 青（クエン酸） | クエン酸Na | 凝固検査（PT、APTT） |
| 緑（ヘパリン） | ヘパリンNa/Li | 緊急生化学 |
| 黒（クエン酸高濃度） | クエン酸Na | 赤沈（ESR） |
| 灰（フッ化Na） | NaF + EDTA | 血糖（FBS） |
| 黄（ACD） | クエン酸・デキストロース | 輸血関連 |

> **採血管の採取順序：**
> 血液培養 → 凝固系（青） → 血清系（赤） → ヘパリン（緑） → EDTA（紫） → フッ化Na（灰）
> ※施設のマニュアルに従ってください。

## 4. 感染予防・標準予防策

### スタンダードプリコーション（標準予防策）
すべての患者の血液・体液を感染性とみなして対応します。

**必須事項：**
1. **手指衛生** — 処置前後に流水または擦式アルコール消毒
2. **個人防護具（PPE）の着用** — 手袋は必須。血液曝露リスクが高い場合はマスク・ゴーグルも検討
3. **針刺し事故防止**
   - リキャップ（針のキャップを戻す行為）は原則禁止
   - 使用した針は直ちに針捨て容器へ
   - 安全機能付き採血針の活用

### 針刺し事故発生時の対応
1. 傷口をすぐに流水で洗い流す（絞り出しは不要）
2. 直ちに上司・医師へ報告
3. 事故報告書の記載
4. 曝露源の患者の感染症検査（同意のもと）
5. 必要に応じてHIV暴露後予防投与（PEP）の検討（2時間以内に開始）

## 5. 採血の手順

### STEP 1：準備

```
□ 指示確認（医師オーダー・検査項目の確認）
□ 検体ラベルの準備と患者情報の照合
□ 必要物品の準備
□ 手指衛生
```

### STEP 2：患者確認・説明

```
□ フルネームで名乗ってもらい本人確認（ネームバンド照合）
□ アレルギー歴・過去の採血での気分不良歴を確認
□ 処置の目的・方法をわかりやすく説明し同意を得る
□ 体位を整える（座位または仰臥位）
```

### STEP 3：穿刺部位の決定

```
□ 袖をまくってもらう
□ 駆血帯を穿刺部位から5〜10 cm中枢側（心臓寄り）に装着
□ 強く締めすぎない（指1本入る程度）
□ 手を軽く握ってもらう（グーパー運動は溶血の原因になるため避ける）
□ 視診・触診で静脈を確認
```

### STEP 4：消毒

```
□ 手袋を着用
□ アルコール綿で穿刺部位を中心から外側へ円を描くように消毒
□ 消毒が乾くまで待つ（10〜30秒）
□ 乾燥前に触れない
```

### STEP 5：穿刺

```
□ 非利き手の親指で皮膚を遠位方向（手首側）に牽引し静脈を固定
□ 針のゲージと刃面（ベベル）を上向きに確認
□ 皮膚に対して15〜30度の角度で穿刺
□ 血液の逆流（フラッシュバック）を確認
□ 角度を下げて静脈に沿って少し進める
```

### STEP 6：採血

```
□ 採血管を規定量になるまでホルダーに押し込む
□ 順序に従って採血管を交換
□ 各採血管を抜いたらすぐに穏やかに転倒混和（6〜8回）
□ ※激しく振ると溶血の原因になる
```

### STEP 7：抜針・止血

```
□ 最後の採血管を外してから駆血帯を外す
□ アルコール綿またはガーゼを穿刺部位に当て
□ 針を素早く抜去
□ 直ちに針捨て容器へ廃棄（リキャップ禁止）
□ 3〜5分間圧迫止血（抗凝固薬服用患者は5〜10分）
□ もんではいけない（血腫・溶血の原因）
□ 止血確認後に絆創膏を貼付
```

### STEP 8：後処理・記録

```
□ 患者の状態確認（気分不良・顔色）
□ 体位を整え、必要があれば安静を促す
□ 検体ラベルを確認し搬送
□ 使用物品を廃棄・整理
□ 手指衛生
□ 電子カルテへの記録（実施時刻、穿刺部位、患者の状態）
```

## 6. 穿刺部位の選択と注意点

### 静脈の触知・視認のコツ

| 静脈が見えにくいとき | 対処法 |
|---------------------|--------|
| 血管が細い・浮きにくい | 腕を下に垂らす、温める（温タオルなど） |
| 肥満で視認困難 | 触診を重視、細いエコーガイドの活用も |
| 脱水患者 | 補液後や水分摂取後に再検討 |
| 高齢者で血管が脆弱 | 細い針（23G以上）を選択、把持力を調整 |

### 針のサイズ（ゲージ）選択の目安

| 状況 | 推奨ゲージ |
|------|-----------|
| 一般成人 | 21G |
| 血管が細い・高齢者 | 22〜23G |
| 大量採血・輸血前 | 18〜20G |

## 7. 合併症と対応

### 主な合併症

| 合併症 | 原因 | 予防・対応 |
|--------|------|-----------|
| **血腫** | 不完全な止血、静脈貫通 | 圧迫止血の徹底、揉まない |
| **神経損傷** | 尺側静脈穿刺時に正中神経を損傷 | 部位の選択、患者の「しびれ・電気が走る感覚」で即座に抜針 |
| **迷走神経反射（血管迷走神経反応）** | 疼痛・恐怖による副交感優位 | 仰臥位での採血推奨、事前に問診、早期のスニッフィングポジション |
| **溶血** | 過度の陰圧、激しい混和、細い針 | 適切な技術の習得 |
| **感染** | 消毒不足、無菌操作の不徹底 | 標準予防策の遵守 |
| **針刺し事故** | リキャップ、不注意な操作 | 安全器材の使用、廃棄ルールの遵守 |

### 迷走神経反射への対応手順
1. 採血を中断し、針を抜く
2. 患者を仰臥位にして足を挙上
3. 衣類を緩め、意識・呼吸・脈拍を確認
4. 冷水タオルを首・額に当てる
5. 改善しない場合は医師へ報告

## 8. 検体の取り扱い

### 溶血を防ぐポイント
- 採血管を激しく振らない → **穏やかに転倒混和（6〜8回）**
- 細すぎる針の使用を避ける
- 採血後すぐに混和する
- 注射器採血の場合は採血管に静かに注入（泡立てない）

### 検体の搬送
| 検査の種類 | 搬送条件 |
|-----------|----------|
| 一般検体 | 常温で速やかに |
| 血液ガス | 氷水中で速やかに（30分以内） |
| 血液培養 | 体温に近い状態で速やかに |
| 凝固検査 | 常温で速やかに（冷蔵不可） |

> **採血後は速やかに検査室へ搬送します。放置時間が長いと検査値に影響します。**

### ラベルの確認
- 患者氏名・ID・生年月日の一致
- 採血管の種類と検査項目の一致
- 採血日時の記録

## 9. 患者への説明と心理的配慮

### 説明のポイント
- 「何のために行うか」を平易な言葉で伝える
- 「どの程度の痛みがあるか」を正直に伝える（「少しチクっとします」）
- 「どのくらい時間がかかるか」を伝える
- 同意を確認してから実施する

### 採血が苦手な患者への配慮
- 仰臥位での採血を提案する
- 会話や深呼吸で注意をそらす
- 「気分が悪くなったら教えてください」と伝える
- 小児には年齢に合わせた声かけと、可能であれば保護者の付き添いを促す

### 高齢患者・認知症患者への配慮
- ゆっくり、はっきりした言葉で繰り返し説明する
- 処置直前にも再度声かけをする
- 安全な体位の確保と固定の工夫

## 10. 演習チェックリスト

以下を自己評価してみましょう（◎できた / ○概ねできた / △要練習）

### 準備・確認フェーズ
```
[ ] 医師の指示・検査項目を正確に確認できた
[ ] 患者をフルネームで確認し、ネームバンドと照合した
[ ] 必要物品を漏れなく準備できた
[ ] 処置前に手指衛生を行った
[ ] アレルギー・採血時の気分不良歴を確認した
```

### 技術フェーズ
```
[ ] 適切な穿刺部位を選択できた
[ ] 駆血帯を適切な位置・圧で装着できた
[ ] 消毒後に乾燥を待ってから穿刺した
[ ] 適切な角度・手技で穿刺できた
[ ] 採血管の順序・本数を正確に守った
[ ] 採血管を穏やかに転倒混和した
[ ] 駆血帯を外してから抜針した
[ ] 針を直ちに針捨て容器に廃棄した（リキャップしなかった）
[ ] 適切な時間・方法で止血した
```

### 後処理フェーズ
```
[ ] 患者の状態（気分不良など）を確認した
[ ] 検体ラベルを確認して速やかに搬送した
[ ] 使用物品を適切に廃棄・整理した
[ ] 処置後に手指衛生を行った
[ ] 電子カルテへ記録した
```

## まとめ

採血は日常的な技術ですが、**感染管理・患者安全・検体の質**に直結する重要な処置です。

**習熟のための3原則：**
1. **正確な知識** — 解剖・検査の意味・合併症を理解する
2. **安全な技術** — 標準予防策と安全器材の活用を徹底する
3. **患者中心の態度** — 説明・同意・心理的配慮を忘れない

> 迷ったとき・困ったときは必ず先輩・指導者に相談してください。
